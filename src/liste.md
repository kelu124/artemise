DÔYE ( n.f. ) : doigt de pied, orteil. J'm'ai fait mal à ma grosse dôye.

DRACHE, DRACHER ( n.f., v.i. ) : pluie d'orage, violente (du néerlandais "draschen" : pleuvoir à verse). T'aurais vu la drache qu'on a ramassée, t' aurais vu comme çà drachait.

ÈCAFILLE, ÈCAFILLER ( n.f., v.t. ) : éclat de coquille d'oeuf, de coque de noix ou de noisette; enlever la coquille, débarrasser les fruits de leurs coques. Quoi qu'on fait des écafilles ? On les met dans l'poêle ?


ÈCADOT ( n.m. ) : ongle du cochon La première chose qu'on mange sur le cochon, c'est la chair qu'on trouve à l'intérieur des écadots. 

ÈTICOT ( n.m. ) : perche, généralement de sorbier (une pousse de l'année) utilisée pour ramer les haricots.

ÈTRUQUER [S'] ( v.p. ) : s'étrangler (en avalant de "travers").
ÈTRICHER [S'] ( v.p. ) : s'étrangler (en avalant de "travers").

GALINER ( v. i. ) : faire des glissades, de la luge La Semoy est prise, on va pouvoir aller galiner, faire la "p'tite grand'mère". Galiner, c'est faire de la sellette sur la neige.

GLÔYE ( n. f. ) : gadoue, caca d'oie, ou d'une autre volaille. te v'là beau, t'as co mis lè pi dans la glôye.

HUCHE ( n.f. ) : porte (de 'huis', porte, voir huisserie, huissier) à l'huche : dehors . Froum' l'huche, qu'lè camps volants n'rent' mi ! Fous çà à l'huche, [sinon] çà va co faire un râlage.


LIPETTE ( n.f. ) : languette de cuir, sous le laçage des chaussures. R'monte bin ta lippette, sinon ça va t'faire mal aux pi'ds.

LOQUE ( n.f. ) : serpillière (wassingue, torchon, bâche, since, panosse, vadrouille, faubert, ... ailleurs en France). Passer la loque [à reloqueter], c'est reloqueter.

MANIQUE ( n.f. ) : poignée de certains ustensiles (anc.fr.: manche de certains outils).
MARÎNDE ( n.f. ) : goûter, casse-croûte des écoliers, encas permettant d'attendre le repas familial (sandwich).

MORCIAU ( n.m. ) : morceau. Pour son goûter, il avait juste un morciau d'pain, un quignon quoi !
